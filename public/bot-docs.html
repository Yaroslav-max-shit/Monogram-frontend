Monogram Bot API

Документация для создания и управления ботами в мессенджере Monogram.

Создание бота

Боты создаются через интерфейс мессенджера (кнопка "Создать бота" в поиске). После создания вы получите:

  • bot_id — уникальный ID бота
  • api_key — секретный ключ для запросов

Юзернейм бота должен заканчиваться на Bot или _bot.

Базовый URL

https://monograme.netlify.app (или https://monogram-backend-dxv4.onrender.com/ для локальной разработки)

Методы API

POST /bots/{bot_id}/send

Отправка сообщения от имени бота.

Параметры

ПараметрТипОписание
api_keystringСекретный ключ бота
chat_idintegerID чата куда отправить
contentstringТекст сообщения

Пример

import requests

requests.post("https://monogram-backend-dxv4.onrender.com//bots/1/send", params={
    "api_key": "ваш_ключ",
    "chat_id": 1,
    "content": "Привет от бота!"
})
GET /bots/{bot_id}/updates

Получение новых сообщений, адресованных боту.

Параметры

ПараметрТипОписание
api_keystringСекретный ключ бота
offsetintegerID последнего полученного обновления (необязательно)

Пример ответа

[
  {
    "update_id": 42,
    "message": {
      "message_id": 42,
      "chat_id": 1,
      "content": "Привет бот!",
      "sender_id": 3,
      "timestamp": "2026-05-15T12:00:00"
    }
  }
]
GET /bots/my

Список моих ботов (требует авторизацию пользователя).

Ограничения

  • Бот может отправлять сообщения только в чаты где состоит его владелец
  • Бот не может инициировать новый чат
  • API ключ действителен пока бот активен

Пример бота

import requests
import time

API_KEY = "ваш_ключ_бота"
BOT_ID = 1
last_update = 0

while True:
    updates = requests.get(
        f"https://monogram-backend-dxv4.onrender.com/bots/{BOT_ID}/updates",
        params={"api_key": API_KEY, "offset": last_update}
    ).json()
    
    for update in updates:
        msg = update["message"]
        chat_id = msg["chat_id"]
        text = msg["content"]
        
        # Ответ пользователю
        requests.post(
            f"https://monogram-backend-dxv4.onrender.com/bots/{BOT_ID}/send",
            params={
                "api_key": API_KEY,
                "chat_id": chat_id,
                "content": f"Вы написали: {text}"
            }
        )
        
        last_update = update["update_id"]
    
    time.sleep(1)
